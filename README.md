# Управление подписками — мобильный PWA-интерфейс

Готовая мобильная часть для MVP сервиса управления подписками. Интерфейс сделан в визуальном стиле банковского приложения: светлый фон, жёлтый акцент, крупные карточки, нижняя навигация, витрина подписок на главной странице, фильтры, экран управления подпиской и отдельная страница профиля.

## Стек

- React
- Vite
- Tailwind CSS
- PWA: `manifest.webmanifest` + `service worker`

## Что реализовано

- Главная страница с сервисом «Управление подписками».
- Общая сумма подписок в месяц.
- Расчёт потенциальной экономии.
- Фильтры: все, активные, возможно лишние, подорожавшие.
- Список подписок с ценой, датой следующего списания и статусом.
- Нижняя навигация без раздела «Чат»: Главная, Платежи, Профиль.
- Отдельная страница профиля пользователя.
- Bottom sheet при нажатии на подписку.
- Действия: отметить как важную, отключить предупреждения, скрыть ошибочно найденную подписку.
- PWA-режим: приложение можно открыть на телефоне и добавить на главный экран.

## Как запустить на компьютере

```bash
npm install
npm run dev
```

После запуска Vite покажет адрес вида:

```bash
Local:   http://localhost:5173/
Network: http://192.168.1.10:5173/
```

## Как открыть на телефоне

1. Компьютер и телефон должны быть в одной Wi‑Fi сети.
2. Запустить проект командой:

```bash
npm run dev
```

3. На телефоне открыть `Network`-адрес, который покажет Vite. Например:

```text
http://192.168.1.10:5173/
```

4. В браузере телефона выбрать «Добавить на главный экран».

## Как собрать production-версию

```bash
npm run build
npm run preview
```

## Где менять моковые данные

Моковые подписки лежат в файле:

```text
src/subscriptions.js
```

В дальнейшем вместо этих данных можно подключить backend на Python/FastAPI.
